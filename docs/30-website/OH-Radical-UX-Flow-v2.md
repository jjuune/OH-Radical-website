# OH Radical — 메인 페이지 UX Flow 전략 문서 v2.0

> **작성 기준:** OH Radical 글로벌 브랜드 플랫폼 전략 문서 + 메인 시안 (2026-0528.png)
> **문서 성격:** MVP 메인 페이지 UX 전략 실무 기획서 (비주얼 방향 v2 반영)
> **핵심 UX 흐름:** 호기심 → 명확함 → 신뢰 → 공감 → 확신 → 행동

---

## 📐 UX 설계 철학 v2: "기술이 조용히 공간을 보호한다"

v1 문서의 핵심 명제가 **"보이지 않는 것을 보이게 하라"** 였다면,  
v2의 핵심 명제는 **"보이지 않아도 이미 신뢰받는다"** 이다.

v1이 "강력한 기술을 드라마틱하게 보여주는 플랫폼"이었다면,  
v2는 **"공간 안에 조용히 존재하는 신뢰의 브랜드"** 다.

방문자가 처음 느껴야 하는 것은 기술력이 아니다.  
**편안함 → 명확함 → 신뢰 → 확신** 의 순서로 감정이 쌓여야 한다.  
기술은 그 신뢰의 근거로서 자연스럽게 뒤따라 등장한다.

시안에서 확인한 비주얼 방향:
- **Bright White Base** — 어둠이 아닌 빛으로 출발
- **Airy Atmosphere** — 공간감, 여백, 공기의 흐름
- **Soft Mint/Cyan Accent** — 기술 포인트는 차갑지 않게
- **Premium Whitespace** — 정보를 담는 것이 아니라 공기를 담는 레이아웃
- **Calm Depth** — 조용하지만 깊이 있는 브랜드 인상

> ⚠️ v2 핵심 원칙: 이 플랫폼은 장비를 파는 사이트가 아니라,  
> **"공간을 위한 미래형 clean-air experience 브랜드"** 를 경험시키는 플랫폼이다.

## Section 0. Global Navigation

### 섹션 목적
라이트 모드 기반의 네비게이션으로, 방문자가 진입하는 순간 "정돈되고 신뢰감 있는 브랜드"임을 0.5초 안에 인식시킨다. 어둡고 무거운 첫인상을 탈피하고, 밝고 정제된 테크 브랜드의 인상을 전달한다.

### 사용자 감정 변화
진입 → "어, 깔끔하다. 믿을 수 있겠는데" → 자연스럽게 아래로 시선 이동

### 핵심 메시지
OH Radical은 정제된 클린에어 브랜드다. 복잡하지 않고, 무겁지 않다.

### 추천 카피 방향
- 로고: `OH` (민트/시안 컬러) + `Radical` (다크 텍스트) — 시안 이미지 방식 그대로
- 메뉴: `Technology` / `Solutions` / `Applications` / `Research` / `About Us`
- 우측: `KR ∨` (언어 전환) + `≡` (햄버거 또는 풀메뉴)

### 추천 인터랙션/모션
- 스크롤 시: 투명 → 흰 배경 + 하단 shadow(0 1px 0 rgba(0,0,0,0.06)) 전환
- 메뉴 Hover: 민트/시안 컬러 텍스트 전환 (subtle, 자연스럽게)
- 로고: 심플한 워드마크, 애니메이션 없음 (MVP)

### 추천 배경 비주얼 무드
- 초기: 완전 투명 (Hero 배경이 비침)
- 스크롤 후: Pure White `#FFFFFF` + 하단 미세 그림자

### 모바일 대응
- 로고 좌 / 언어 전환 + 햄버거 우
- 풀스크린 흰 오버레이 네비

## Section 1. Hero — "Invisible Protection. Visible Trust."

### 섹션 목적
시안 이미지의 Hero 구조를 기준으로, 방문자를 "밝고 깨끗한 공간"으로 초대한다. 이전 v1처럼 기술을 드라마틱하게 등장시키는 것이 아니라, 이미 정화된 공간 속에 방문자가 자연스럽게 들어선 느낌을 준다. 브랜드가 먼저 말하지 않는다. 공간이 먼저 느껴진다.

### 사용자 감정 변화
"어, 깨끗하다" → "이게 뭔 회사지?" → "공기 기술이구나" → "더 알아보고 싶다"

### 핵심 메시지
> **We Engineer the Air You Trust.**  
> 보이지 않는 기술로, 당신이 신뢰할 수 있는 공기를 만듭니다.

### 추천 카피 방향

**키커 (상단 소형 레이블):**
```
Invisible Protection. Visible Trust.
```
→ 시안에서 확인된 그대로. 민트 컬러 텍스트, 소형 (14px)

**메인 헤드라인 (대형):**
```
We Engineer
the Air You Trust
```
→ "Air"는 민트/시안 컬러 강조 (시안 이미지 방식 그대로)
→ 무게감 있는 Sans-Serif, 다크 텍스트 (#1a1a2e 또는 #0d1117)
→ 크기: 72~80px (desktop)

**서브 카피:**
```
OH Radical은 보이지 않는 기술로
공기 중 유해물질과 바이러스를 근본적으로 분해하여
더 안전하고 건강한 공간을 만듭니다.
```
→ 작고 차분하게. 16px, 색상 #555, 줄간격 1.8

**CTA:**
```
[Discover Technology  >]    ▶  Watch Video
```
→ 시안 그대로: Outline 버튼 + 텍스트 링크 병렬 배치

### 추천 인터랙션/모션

**[배경 비주얼 구성 — 시안 기준]**
- 우측 절반: 밝고 투명한 클린 인테리어 공간 이미지 (흰 공간, 자연광)
- 그 위에: 기기(OH Radical 장비) 반투명 합성
- 기기 주변: 부드러운 민트 색상 기류 흐름 애니메이션 (Soft Wave SVG)
- OH 분자 뱃지: 민트 원형 배지, 공중에 부유 (gentle float 애니메이션)
- 배경에 미세 파티클: 흰 공간 속 매우 연한 민트 점들

**[진입 애니메이션 — MVP 기준]**
- 배경: 이미지 자체는 정적 (초기 로드 안정성 우선)
- 키커 → 헤드라인 → 서브카피 → CTA 순서 Fade In (Stagger 0.3s)
- 기류 애니메이션: CSS keyframe 기반 SVG 곡선 (Three.js 미사용)

### 추천 배경 비주얼 무드
- **전체 배경:** 밝은 흰색 베이스 `#f8fafb` ~ `#ffffff`
- **좌측 영역:** 순수 흰 여백 (텍스트 가독성 최우선)
- **우측 영역:** 투명한 클린 공간 이미지 + 기류 시각화 레이어
- **전체 분위기:** 병원도 공장도 아닌, 고급 건축 공간 같은 느낌 (Aesop 쇼룸, Apple Store)
- **빛의 방향:** 오른쪽 상단에서 자연광이 들어오는 느낌

### CTA 방향
- Primary: `Discover Technology` (아웃라인 버튼, 다크 텍스트)
- Secondary: `▶ Watch Video` (텍스트 링크)
- 공격적이지 않게. 탐색을 유도하는 방식.

### 다음 섹션으로 이어지는 UX 연결 논리
방문자는 "이 공간이 왜 이렇게 깨끗하지?"를 느꼈다.  
아래로 스크롤하면 Hero 하단에 붙어있는 **4-Concept Bar**가 기술을 4단어로 압축해서 보여주며 "이 기술이 어떻게 작동하는지"에 대한 첫 이해를 제공한다.

## Section 2. 4-Concept Bar — "기술의 4단계 요약"

### 섹션 목적
시안에서 확인된 핵심 구성 요소: Hero 바로 아래 4개의 핵심 기술 개념을 가로로 나열하는 컴팩트 섹션. 기술을 길게 설명하지 않고, 4개의 단어+한 줄 설명으로 브랜드 기술 프레임을 즉시 각인시킨다. 방문자에게 "이 기술이 뭘 하는지" 30초 안에 이해시키는 것이 목적.

### 사용자 감정 변화
"뭘 하는 회사인지 딱 보이네" → 명확함 → "더 알아보고 싶다"

### 핵심 메시지
> 분해하고, 정화하고, 전환하고, 보호한다.

### 추천 카피 방향

```
[아이콘] Decompose          [아이콘] Purify
        유해물질·바이러스 분해        공기질 실시간 정화
        Learn more >                 Learn more >

[아이콘] Convert            [아이콘] Protect
        물(H₂O)로 완전히 전환        지속 가능한 안심 공간
        Learn more >                 Learn more >
```

→ 시안의 4개 카드 구조 그대로  
→ 아이콘: 선형 아이콘 (민트/시안 포인트)  
→ `Learn more >` : 텍스트 링크, 해당 기술 앵커 또는 Technology 페이지 연결

### 추천 인터랙션/모션
- Hover: 카드 배경이 매우 연한 민트 틴트로 전환 (0.2s)
- 카드 하단 `Learn more` 화살표: hover 시 우측으로 3px 이동
- MVP: CSS hover만으로 충분

### 추천 배경 비주얼 무드
- 배경: Hero보다 약간 더 밝거나 동일 (`#ffffff` 또는 `#f7f9fb`)
- 카드 구분: 세로 0.5px 선 `rgba(0,0,0,0.08)`
- 전체 섹션 높이: 160~200px (너무 높지 않게)
- 아이콘: 작고 정제된 선형 아이콘 (Lucide React 또는 커스텀)

### CTA 방향
각 카드의 `Learn more >` — 공격적 CTA 아님. 탐색 유도.

### 다음 섹션으로 이어지는 UX 연결 논리
기술을 4개 단어로 이해했다. 이제 방문자는 "근데 내 공간에도 이게 필요한가?"를 자문한다.  
→ **Why Clean Air** 섹션이 그 질문에 공감으로 답한다.

## Section 3. Why Clean Air — "공기를 다시 생각하다"

### 섹션 목적
v1의 "위험 체감" 섹션을 "공감과 인식" 중심으로 전환한다. 위협을 강조하는 것이 아니라, "우리가 매일 머무는 공간의 공기는 얼마나 건강한가?"라는 질문에서 출발한다. 공포가 아닌 인식 전환. 브랜드가 걱정을 주는 것이 아니라, 새로운 시각을 주는 것이다.

### 사용자 감정 변화
"생각해보니 공기를 신경 쓴 적이 없었네" → 공감 → "이게 중요하구나" → 관심

### 핵심 메시지
> 당신이 머무는 공간의 공기,  
> 생각보다 많은 것들이 조용히 떠다닙니다.

### 추천 카피 방향

**섹션 타이틀:**
```
The Air Around You.
우리가 숨쉬는 공기에 대하여.
```

**각 공기질 이슈 접근 방식 — "위협"이 아닌 "사실"로 서술:**

| 항목 | 헤드라인 | 본문 톤 |
|---|---|---|
| 바이러스 | "Airborne viruses spread without warning." | "밀폐 공간에서 공기 중 바이러스는 눈에 보이지 않습니다. 기존 환기로는 근본 해결이 어렵습니다." |
| VOC | "New buildings off-gas for months." | "새 공간의 포름알데히드와 VOC는 수개월간 지속됩니다. 대부분 인식하지 못한 채 흡입됩니다." |
| 악취 | "Odor signals chemical imbalance." | "불쾌한 냄새는 단순 불편함이 아닙니다. 공간의 화학적 불균형 신호입니다." |
| 조리흄 | "Cooking fumes are more than smoke." | "급식실과 주방의 조리흄은 단순 연기가 아닙니다. WHO는 이를 1급 발암물질로 분류합니다." |
| 미세먼지 | "PM2.5 finds its way inside." | "창문을 닫아도 초미세먼지는 유입됩니다. 실내 농도가 실외보다 높아지는 경우도 있습니다." |

→ 어조: 차분하고 명확하게. 공포 자극 없이 사실만 전달.  
→ 숫자와 수치는 최소 사용 (신뢰 느낌이지 위협 느낌 아님)

### 추천 인터랙션/모션

**[Sticky Scroll — 밝은 버전]**
- v1과 동일한 Sticky Scroll 구조 유지
- 단, 배경은 밝고 깨끗하게 (다크 배경 → 라이트 배경)
- 좌: 텍스트 / 우: 해당 공기 이슈 시각화 (밝은 공간 이미지 + 연한 컬러 오버레이)

**비주얼 방식 변경:**
- v1: 입자가 탁하게 쌓이는 불안한 시각화
- v2: 깨끗한 공간 사진에 연한 아이콘/일러스트 오버레이 (부드럽게)
- 각 항목 전환: opacity + translateY 부드러운 페이드

**진행 인디케이터:**
- 우측 세로에 얇은 선 + 현재 위치 표시 (01 / 05)

### 추천 배경 비주얼 무드
- 배경 전반: 밝은 `#f0f4f8` ~ `#ffffff`
- 우측 이미지: 각 공간 환경 사진 (오피스, 학교, 병원 등 밝은 분위기)
- 컬러 포인트: 각 항목별 아주 연한 민트/시안 아이콘 또는 선 그래픽
- 전체 무드: 불안 유발 금지 — 차분하고 인식을 확장시키는 느낌

### CTA 방향
섹션 하단:
```
OH Radical은 이 공간들을 어떻게 바꾸는가.
→ (자연스러운 스크롤 연결)
```

### 다음 섹션으로 이어지는 UX 연결 논리
방문자가 공기 인식을 얻었다. "그래서 OH Radical이 뭘 어떻게 해결하는 거지?"  
→ **Technology 섹션**이 기술 원리를 아름답게 보여준다.

## Section 4. Technology — "이해 가능하고 아름다운 과학"

### 섹션 목적
기술 원리를 드라마틱하게 펼치는 것이 아니라, 이해 가능하고 아름답게 보여준다. "복잡하고 강력한 기술"이 아니라 "투명하고 명확한 기술"로 포지셔닝. 방문자가 기술을 이해하는 순간 신뢰가 생긴다.

### 사용자 감정 변화
"어떻게 작동하는 거지?" → "아, 이런 방식이구나" → 이해 → "이 회사 기술 진짜네" → 신뢰

### 핵심 메시지
> OH Radical은 자연의 원리로 공기를 정화한다.  
> 반응의 시작은 분자, 결과는 순수한 물(H₂O).

- 전체 사이트의 "조용한 마무리" — 브랜드 컬러 계열로 통일

---

## 🔄 전체 감정 여정 요약 (v2)

```
Section 0 (Nav)          →  신뢰, 정돈된 첫인상
Section 1 (Hero)         →  편안함, 호기심 "깨끗하다. 뭔 회사지?"
Section 2 (Concept Bar)  →  명확함 "이 기술이 이걸 하는구나"
Section 3 (Why Air)      →  공감 "공기를 한 번도 신경 안 썼었네"
Section 4 (Technology)   →  이해 → 신뢰 "과학이 아름답게 작동하는구나"
Section 5 (Solutions)    →  관련성 "내 공간에도 이게 필요하다"
Section 6 (Proof)        →  확신 "검증됐구나, 믿을 수 있다"
Section 7 (Philosophy)   →  공감 "이 브랜드 가치관이 좋다"
Section 8 (Final CTA)    →  결심 "한번 이야기해보자"
```

## 📌 v2 브랜드 임프레션 체크리스트

방문자가 사이트를 떠날 때 이렇게 느껴야 한다:

```
✅ 기술적으로 진보했다
✅ 조용하고 우아하다
✅ 사람을 위한 기술이다
✅ 믿을 수 있는 브랜드다
✅ 내 공간에 어울린다

❌ 산업용 장비 회사 같다
❌ 너무 과학적이고 어렵다
❌ 다크하고 무겁다
❌ SF 영화 같다
❌ 장비를 팔려고 한다
```
