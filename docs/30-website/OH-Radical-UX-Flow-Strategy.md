# OH Radical — 메인 페이지 UX Flow 전략 문서

> **작성 기준:** OH Radical 글로벌 브랜드 플랫폼 구축 기획안 (project-overview.md / brand-core.md / README.md)  
> **문서 성격:** MVP 메인 페이지 UX 전략 실무 기획서  
> **핵심 UX 흐름:** 문제 인식 → 위험 체감 → 기술 이해 → 안전성 검증 → 신뢰 확보 → 문의 전환

---

## 📐 UX 설계 철학: "보이지 않는 것을 보이게 하라"

OH Radical의 기술적 본질은 역설이다.  
가장 강력한 위협(바이러스, VOC, 악취 분자)은 눈에 보이지 않고,  
가장 강력한 솔루션(OH Radical 분자 반응)도 눈에 보이지 않는다.

이 플랫폼의 UX가 해야 할 일은 단 하나다.  
**"보이지 않는 것에 대한 두려움 → 보이지 않는 것에 대한 신뢰"** 로의 감정 여정 설계.

스크롤 한 번 한 번이 "이해의 단계"이자 "신뢰 축적의 단계"여야 하며,  
방문자가 페이지를 다 내려갔을 때 자연스럽게 문의 버튼을 누르고 싶어지도록 설계한다.

> ⚠️ 절대 원칙: 제품을 파는 사이트가 아니라, **세계관을 체험시키는 플랫폼**을 만든다.

## Section 0. Global Navigation

### 섹션 목적
방문자가 사이트에 처음 진입하는 순간, 글로벌 하이엔드 테크 브랜드라는 첫인상을 0.5초 안에 각인시킨다. 내비게이션 자체가 브랜드의 수준을 말해준다.

### 사용자 감정 변화
막연한 기대감 → "이 회사, 뭔가 다르다"는 직관적 인식

### 핵심 메시지
OH Radical은 장비 회사가 아니라 환경 기술 브랜드다.

### 추천 카피 방향
- 로고: `OH Radical` (단독 워드마크, 과학 기호 감성 서체)
- 메뉴 구조: `Technology` / `Solutions` / `Science` / `About` / `Contact`
- 언어 전환: `KO / EN` (글로벌 대응 시각화)

### 추천 인터랙션/모션
- 스크롤 다운 시: 네비게이션이 반투명 배경(Glassmorphism)으로 전환되며 고정 (Sticky)
- 로고 Hover: OH Radical 문자 주변에 극도로 미세한 분자 입자가 잠깐 스파크처럼 등장하다 사라짐
- 메뉴 항목 Hover: 하단에 Cyan/Mint 계열의 극세선 언더바 슬라이드 등장

### 추천 배경 비주얼 무드
- 초기 상태(Hero 배경이 비침): 반투명 다크 (rgba 블러)
- 스크롤 후: 완전 불투명 딥 다크 + 백색 로고

### CTA 방향
우상단에 `문의하기 / Get in Touch` 버튼 고정 노출. 어느 섹션에서든 전환 경로를 열어둔다.

### 다음 섹션으로 이어지는 UX 연결 논리
네비게이션은 Hero로 진입하는 "문"이다. 잘 만들어진 내비는 방문자에게 "내가 어디 와있는가"를 알려주는 동시에, 아래 세계관으로의 진입 욕구를 자극한다.

## Section 1. Hero Section — "Invisible. Unstoppable."

### 섹션 목적
브랜드의 세계관을 1~3초 안에 압도적으로 전달한다. 방문자가 스크롤을 멈추고 이 브랜드가 무엇인지를 느끼는 유일한 순간이다. 설명하지 않는다. 체험시킨다.

### 사용자 감정 변화
"어? 이게 환경 장비 회사야?" → 경이로움 → "계속 보고 싶다"

### 핵심 메시지
> **"Human First. Earth Always."**  
> 기술이 보이지 않아도, 당신은 이미 보호받고 있습니다.

### 추천 카피 방향

**메인 헤드라인 (대형 타이포):**
```
Human First.
Earth Always.
```

**서브 카피:**
```
The invisible defense system.
보이지 않는 곳에서 작동하는 인류의 안전 인프라.
```

**보조 설명 (선택):**
```
OH Radical 기술은 공기 중의 바이러스, 유해 가스, 악취 분자를 
화학약품 없이, 필터 없이, 폐기물 없이 파괴합니다.
```

### 추천 인터랙션/모션

**[Full-Screen Cinematic Entry]**
- 화면 전체를 채우는 어두운 공간(Deep Space / 실내 공기 공간 은유)
- 카메라가 서서히 당기면서(Zoom Out) 보이지 않던 공기 속 입자들이 나타나기 시작
- 수백 개의 미세 입자(바이러스, 먼지, 분자)가 공간에 떠다니는 시각화
- 약 2초 후: 화면 중앙에서 OH Radical 분자(·OH)가 생성되며 빛을 발산
- 그 빛이 퍼지면서 입자들이 하나씩 소멸 (물 분자로 변환되는 애니메이션)
- 소멸 후 공간이 완전히 투명하고 맑아짐 → 공기가 정화된 상태 표현

**[Scroll-Triggered Progression]**
- 스크롤 시작 시: 헤드라인 타이포가 하늘에서 떨어지듯 등장 (Gravity Drop)
- 헤드라인 등장 후: CTA 버튼이 부드럽게 Fade In

**[Background Particle System]**
- 마우스 무브먼트에 반응하는 미세 입자 필드 (Vanilla JS Canvas 또는 Three.js)
- 마우스 주변 입자가 미세하게 밀려나는 인터랙션 (마치 숨쉬는 공간 표현)

### 추천 배경 비주얼 무드
- **컬러:** Deep Space Black (#040a0f) 베이스
- **입자:** 극히 낮은 채도의 Cyan/Blue 계열 마이크로 파티클 (≈ 투명한 공기 속 부유물)
- **빛:** OH Radical 분자가 등장할 때만 Mint/Bio-Cyan 계열 점광원 발산
- **텍스트:** Pure White, 대형 세리프 계열 또는 과학 논문 감성 모노스페이스 혼용
- **전체 무드:** 우주를 담은 현미경 이미지처럼 — 광대하지만 정밀한 느낌

### CTA 방향
- `솔루션 탐색하기` (Primary, Outline/Ghost 버튼 → 흰 테두리 + 내부 투명)
- `기술 알아보기` (Secondary, 텍스트 링크 + 미세 화살표 아이콘)

> CTA는 공격적이지 않게. 이 섹션은 브랜드가 말을 건네는 시간이 아니라, 방문자가 브랜드에 흡수되는 시간이다.

### 다음 섹션으로 이어지는 UX 연결 논리
방문자는 "아름답고 신비로운 무언가"를 봤다. 이제 궁금증이 생긴다.  
**"근데 이게 왜 필요한 거지?"** 라는 질문에 자연스럽게 Why It Matters가 답한다.  
Hero 하단에 아주 작은 다운 스크롤 인디케이터(점 3개 or 선)를 배치해 자연스럽게 유인한다.

## Section 2. Why It Matters — "당신이 모르는 공기 속의 전쟁"

### 섹션 목적
방문자에게 "우리는 사실 보이지 않는 위험 속에 살고 있다"는 사실을 데이터와 시각화로 각인시킨다. 불안감을 조장하는 것이 아니라, 현실을 직시하게 하는 것이다. 이 섹션은 감정적 공감과 이성적 이해가 동시에 일어나야 한다.

### 사용자 감정 변화
무관심 → "생각보다 심각하네" → 공감 → "이걸 해결할 방법이 있을까?"

### 핵심 메시지
> 보이지 않기 때문에 더 위험하다.  
> 당신의 공간 속에는 지금 이 순간도 바이러스, 독성 가스, 악취 분자가 존재한다.

### 추천 카피 방향

**섹션 타이틀 (대형):**
```
What's in your air?
지금 이 순간, 당신이 숨쉬는 공기 속에는.
```

**각 위험 요소 카드 카피 방향:**

| 위험 요소 | 헤드라인 카피 | 서브 카피 |
|---|---|---|
| 🦠 바이러스 | "The invisible epidemic" | 밀폐 공간 내 1인 감염이 수십 명으로 확산되기까지 불과 몇 시간. 공기 중 부유 바이러스는 기존 환기로 제거되지 않는다. |
| ⚗️ VOC | "Your building is breathing poison" | 새 건물, 새 가구에서 방출되는 포름알데히드. 눈에 보이지 않지만 매일 흡입된다. |
| 💨 악취 | "Odor is not just a feeling" | 암모니아와 복합 악취 분자는 공간의 생산성과 거주자의 심리를 직접 훼손한다. |
| 🍳 조리흄 | "The kitchen is a cancer risk" | 급식실, 산업 조리시설의 조리매연은 WHO 지정 1급 발암물질. 환기만으로 해결되지 않는다. |
| 🌫️ 미세먼지 | "You can't close the door on PM2.5" | 외부 미세먼지는 환기구, 문틈을 통해 실내로 유입되어 지속적으로 축적된다. |

### 추천 인터랙션/모션

**[Horizontal Scroll Panel 또는 Sticky Reveal Scroll]**

**방식 A — Horizontal Scroll (권장):**
- 세로 스크롤 시, 섹션 내부는 가로로 패널이 이동하는 방식
- 각 위험 요소가 하나씩 등장 (마치 현미경 슬라이드를 넘기는 느낌)
- 각 패널마다 해당 입자/분자의 3D 시각화가 배경에서 움직임

**방식 B — Sticky Scroll with Progress:**
- 섹션이 Sticky로 고정된 상태에서 스크롤할 때마다 위험 요소가 하나씩 교체
- 좌측: 텍스트 카피 / 우측: 시각화 애니메이션
- 하단에 Progress Bar 또는 숫자 인디케이터 (1/5, 2/5...)

**각 위험 요소 등장 방식:**
- 공간이 깨끗한 상태에서 → 점점 해당 입자가 퍼져나가는 애니메이션
- 입자 밀도가 증가하면서 수치 카운터 상승 (CFU/m³ 등 과학 단위)

### 추천 배경 비주얼 무드
- 배경: 실내 공간 (흐릿한 오피스, 병원 복도, 학교 교실) 느낌의 미묘한 배경
- 초기: 청정하게 보이는 공간
- 스크롤 진행: 점점 입자가 쌓이며 공간이 탁해지는 변화
- 각 위험 요소: 해당 분자의 색상으로 표현 (바이러스=붉은빛, VOC=황색, 미세먼지=회색)

### CTA 방향
이 섹션에는 직접적인 CTA 없음. 대신 섹션 하단에:
```
이 위협들을, 화학약품 없이 제거할 수 있다면?
→ (스크롤 다운 인디케이터)
```
방문자 스스로 스크롤할 동기를 부여한다.

### 다음 섹션으로 이어지는 UX 연결 논리
방문자는 이제 "문제"를 체감했다. 불안하지만 궁금하다.  
다음 섹션은 그 해결의 핵심 원칙인 **3무 시스템**을 제시하며 안도감을 준다.  
감정 흐름: 불안 → 안도 전환이 이 섹션 → 다음 섹션의 핵심 여정이다.

## Section 3. 3무 시스템 — "없앨수록 강해진다"

### 섹션 목적
OH Radical 기술의 근본적 차별점을 "있는 것"이 아닌 "없는 것"으로 정의하는 역발상 설계. 기존 경쟁사들이 "우리는 이것도 있고 저것도 있다"고 외칠 때, OH Radical은 "우리는 이것도 없고 저것도 없다"고 말한다. 무필터, 무화학약품, 무폐기물 — 이 세 가지 부재(不在)가 경쟁 우위임을 납득시킨다.

### 사용자 감정 변화
"이게 없어도 되는 건가?" → "없는 게 맞는 거네" → "이게 진짜 친환경이구나" → 납득과 안도

### 핵심 메시지
> 가장 강력한 정화 기술은 아무것도 소모하지 않는다.  
> 필터도, 약품도, 폐기물도 없다. 오직 OH Radical만 남는다.

### 추천 카피 방향

**섹션 타이틀:**
```
Less is the technology.
없앨수록, 더 강해집니다.
```

**각 시스템 카피:**

**① 무필터 (Filter-Free)**
```
Headline: "No filter. No compromise."
Body: 필터는 오염을 모으는 것이지, 파괴하는 것이 아닙니다.
      OH Radical은 필터 없이, 공기를 공기 그 자체로 정화합니다.
      유지비 zero. 관리 부담 zero.
```

**② 무화학약품 (Chemical-Free)**
```
Headline: "Pure protection. Zero chemistry."
Body: 락스, 알코올, 소독제는 인체에도 반응합니다.
      OH Radical은 자연 상태의 활성 산소만을 사용하며,
      최종 반응물은 순수한 물(H₂O)입니다.
```

**③ 무폐기물 (Zero Waste)**
```
Headline: "The only exhaust is clean air."
Body: 필터 폐기, 독성 폐수, 탄소 배출 — 없습니다.
      ESG 목표를 충족하는 가장 실질적인 방법.
```

### 추천 인터랙션/모션

**[3-Step Sticky Reveal with Visual Transformation]**

각 무(無)에 해당하는 비주얼이 스크롤에 따라 변환:

- **무필터:** 기존 HEPA 필터 구조 3D 이미지 → 스크롤 → 구조가 서서히 사라지고 깨끗한 기류만 남음
- **무화학약품:** 소독액 분사 → 스크롤 → 사라지고, 맑은 H₂O 수분 입자만 남으며 공간이 맑아짐
- **무폐기물:** 쓰레기 더미 → 스크롤 → 완전 소멸 + 녹색 리프 아이콘 등장

**[숫자 카운터 애니메이션]**
각 무(無) 항목 등장 시 유지보수 비용 절감 퍼센티지, 탄소 배출 감소 수치 등이 카운팅되며 올라가는 효과.

**[배지 스타일 강조]**
각 항목에 `❌ Filter` `❌ Chemical` `❌ Waste` 배지를 타이포그래피로 강렬하게 표기 — 지워진 선을 통해 "없음"을 시각화

## Section 4. Technology Preview — "분자 단위의 혁명"

## Section 6. Brand Philosophy — "우리가 믿는 것"

### 섹션 목적

모든 UX 여정의 종착점. 방문자가 지금까지 쌓아온 이해, 공감, 신뢰를 행동으로 전환하는 순간이다. 이 섹션은 "강요"가 아니라 "초대"여야 한다. 이미 설득은 끝났다. 이제 문을 열어주면 된다.

### 사용자 감정 변화
"도입을 검토해봐야겠다" → "어떻게 시작하면 되지?" → 결심 → 액션

### 핵심 메시지
> 당신의 공간을 위한 솔루션은 이미 준비돼 있습니다.

### 추천 카피 방향

**섹션 타이틀 (간결하고 강력하게):**
```
Start the Conversation.
지금, 첫 대화를 시작하세요.
```

**서브 카피:**
```
공간의 규모, 산업 분야, 도입 예산에 관계없이
OH Radical 전문가가 최적의 솔루션을 제안합니다.
```

**CTA 버튼 3개 (행동 분기):**
```
📋 도입 문의하기           →  Contact Form
📄 기술 자료 다운로드     →  Whitepaper PDF
🎯 솔루션 시뮬레이터      →  Interactive Tool (Phase 2)
```

**신뢰 보강 문구 (CTA 하단 소형):**
```
✓ 무료 현장 컨설팅 제공    ✓ 7일 내 담당자 연결    ✓ 인천공항 납품 레퍼런스
```

### 추천 인터랙션/모션

**[Full-Screen Immersive Background]**
- Hero와 유사하지만 반대 방향 — 지금은 모든 공간이 이미 정화된 상태
- 맑고 투명한 공간 + OH Radical 입자가 조용히 작동 중인 비주얼
- 배경이 천천히 밝아지며 "정화 완료" 상태를 상징

**[Contact Form Micro-UX]**
- 폼이 등장할 때: 각 필드가 위에서 부드럽게 슬라이드 다운
- 입력 Focus 시: 필드 하단에 Mint Cyan 라인이 생기며 활성화
- 전송 버튼: Hover 시 배경에 미세 입자 퍼짐 효과

### 추천 배경 비주얼 무드
- 배경: 완전히 투명하고 맑은 공간 (Hero의 오염된 공기와 대비)
- 조용한 입자들이 질서 있게 정화된 공간을 유지하는 느낌
- 컬러: 따뜻한 White + Bio Cyan 포인트
- 전체 무드: 해결된 세계. 평화롭고 청정한 미래.

### CTA 방향
이 섹션 자체가 CTA다.

### 다음 섹션으로 이어지는 UX 연결 논리
CTA 이후 Footer가 이어지며, 세부 정보 탐색과 글로벌 접근성을 제공한다.

## Section 9. Footer

### 섹션 목적
SEO 최적화 + 글로벌 정보 접근성 + 브랜드 완결성 확보.

### 핵심 구성 요소
- 로고 + 슬로건 (`Human First. Earth Always.`)
- 메뉴 구조: Technology / Solutions / About / Contact / Privacy
- 소셜 링크 (LinkedIn, YouTube 등 B2B 채널 중심)
- 다국어 전환 (KO / EN / 향후 JP, ZH)
- 주소 / 사업자정보 (한국 법적 요건)
- 뉴스레터 구독 (GEO 콘텐츠 마케팅 연계)

### 추천 비주얼 무드
- 딥 다크 네이비 배경 + 화이트 텍스트
- 미세한 OH Radical 입자 백그라운드 패턴 (매우 낮은 opacity)
- 전체 사이트의 "엔딩 크레딧"처럼 묵직하고 깔끔하게

## 🔄 전체 감정 여정 요약

```
Section 1 (Hero)          →  경이로움, 압도감, "이게 뭐지?"
Section 2 (Why It Matters)  →  불안, 현실 자각, "생각보다 심각하네"
Section 3 (3무 시스템)    →  안도, 납득, "이런 해결책이 있구나"
Section 4 (Technology)    →  감탄, 확신, "진짜 기술이 있네"
Section 5 (Proof & Data)  →  신뢰, 안심, "데이터가 증명하네"
Section 6 (Philosophy)    →  공감, 감동, "이 회사, 믿을 수 있다"
Section 7 (Solutions)     →  발견, 관심, "나에게도 맞는 게 있네"
Section 8 (Final CTA)     →  결심, 액션, "지금 문의해보자"
```

## 📌 UX 설계 시 주의사항 (Anti-Pattern)

| ❌ 피해야 할 것 | ✅ 대신 이렇게 |
|---|---|
| 제품 스펙 나열 | 기술 원리 + 현장 결과 중심 스토리 |
| "저렴한 가격" 강조 | "비용 절감"의 장기 가치로 프레이밍 |
| 인증서 나열 | 실증 사례와 함께 맥락화하여 제시 |
| 장비 외형 중심 사진 | 기술이 작동하는 공간과 사람 중심 비주얼 |
| 텍스트 과다 | 한 섹션에 핵심 메시지 1개만 |
| 모든 CTA에 "지금 구매" | 단계에 맞는 CTA (탐색→이해→문의) |
| 로딩 지연 방치 | 모션 로딩 인디케이터 + LCP 최적화 필수 |
